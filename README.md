# Regressao Linear Simples e Multipla com Python
Este trabalho tem como finalidade aplicar conceitos fundamentais de regressão linear, análise de dados e avaliação de modelos preditivos utilizando Python e bibliotecas. O projeto explora tanto regressão linear simples quanto regressão linear múltipla, com foco em modelagem matemática, visualização gráfica e validação do desempenho dos modelos. 

## Objetivos:
- Aplicar conceitos teóricos de aprendizado supervisionado
- Implementar modelos de regressão linear a partir da formulação matemática
- Utilizar métodos de mínimos quadrados para estimativa dos parâmetros
- Avaliar o desempenho dos modelos com métricas estatísticas
- Visualizar dados e resultados por meio de gráficos 2D e 3D

## Conteúdos Abordados:
- Regressão Linear Simples
- Regressão Linear Múltipla
- Formulação matricial do modelo
- Estimativa de parâmetros usando:
- Inversa de matriz
- Pseudo-inversa
- Separação de dados em treino e teste
- Avaliação de modelos com métricas estatísticas:
- SSE (Sum of Squared Errors)
- MSE (Mean Squared Error)

## Descrição das Atividades
### Regressão Linear Simples (atividade_regressao.py)
- Modelagem da relação entre duas variáveis
- Construção manual da matriz de projeto
- Cálculo dos coeficientes do modelo via mínimos quadrados
- Visualização da reta ajustada aos dados
- Avaliação do modelo com SSE, MSE e R²

### Regressão Linear Múltipla (pratica_solubilidade.py)
- Modelagem com múltiplas variáveis independentes
- Análise da solubilidade em função de propriedades químicas
- Visualização dos dados em gráfico 3D
- Treinamento e teste do modelo
- Avaliação estatística do desempenho preditivo
