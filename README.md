# Regressao Linear Simples e Multipla com Python
Este trabalho tem como finalidade aplicar conceitos fundamentais de regressão linear, análise de dados e avaliação de modelos preditivos utilizando Python e bibliotecas. O projeto explora tanto regressão linear simples quanto regressão linear múltipla, com foco em modelagem matemática, visualização gráfica e validação do desempenho dos modelos.
